**Portfolio — Designer & Developer**
A sophisticated, single-page portfolio website designed to showcase a dual identity in both design and development. The project features a highly interactive user interface with custom cursors, dynamic themes, and a responsive layout.

🚀 Features
Dual Identity Hero Section: A split-screen hero area that shifts styles based on whether the user interacts with the "Design" or "Developer" side.

Custom Interactive Cursor: A dynamic cursor system that changes shape, color, and size depending on the content it hovers over (e.g., golden square for design, green bar for dev).

Dark Mode Support: A built-in dark mode with automatic adaptation for widgets and UI elements.

GitHub Status Integration: A dedicated widget to display real-time status or project activity, integrated with theme-switching logic.

Modern Typography: Utilizes a combination of Playfair Display (serif), JetBrains Mono (monospace), and DM Sans (sans-serif) to create a professional aesthetic.

🛠️ Built With
HTML5: Semantic structure for accessibility and SEO.

CSS3 (Custom Properties): Extensive use of CSS variables for theming, transitions, and layout management.

JavaScript (ES6+): Powers the custom cursor logic, the dark mode observer, and interactive hover states.

Google Fonts: For high-quality, web-optimized typography.

📂 Project Structure
portfolio.html: The core file containing the structure, styles (CSS), and logic (JS).

🎨 Customization
The project uses CSS Custom Properties (variables) defined in the :root selector for easy global changes:

--accent-design: Golden accent used for design-related elements (#C8A882).

--accent-dev: Green accent used for development-related elements (#4AF7C0).

--serif / --mono / --sans: Easily swap out font families across the entire site.
